## Code example:

### Basic Mathematical Operations

```
function basicOp(operation, value1, value2)
{
  if (operation === '+') {
    return value1 + value2;
  }

  if (operation === '-') {
    return value1 - value2;
  }

  if (operation === '/') {
    return value1 / value2;
  }

  if (operation === '*') {
    return value1 * value2;
  }
}
```
